## 🆕 本次新增功能：分享系统

### 功能清单
- ✅ 创建分享链接（支持权限设置：仅查看/可复制）
- ✅ 分享链接有效期设置（1天/7天/30天/1年）
- ✅ 通过分享链接查看图谱
- ✅ 复制分享的图谱到自己的账户
- ✅ 我的分享列表管理
- ✅ 取消分享功能

### 技术实现
- **后端**: 新增 `share.ts` 路由，数据库表 `GraphShare`
- **前端**: 新增 `SharePanel.tsx` 组件，`SharedGraphView.tsx` 页面
- **路由**: 集成 `react-router-dom`，支持 `/share/:token` 访问

### API 端点
```
POST   /api/share/:graphId/share      # 创建分享
GET    /api/share/token/:shareToken   # 查看分享图谱
POST   /api/share/token/:shareToken/clone  # 复制图谱
GET    /api/share/my/shares           # 我的分享列表
DELETE /api/share/:shareId            # 取消分享
```

## 🔍 本次增强功能：搜索功能完善

### 新增功能
- ✅ **键盘快捷键支持**: ↑↓ 选择结果, ↵ 确认, ESC 关闭
- ✅ **排序选项**: 按相关度/标题/最近更新排序
- ✅ **选中高亮**: 键盘导航时高亮当前选中项
- ✅ **智能排序**: 标题匹配优先，关键词次之，内容最后
- ✅ **搜索历史**: 本地存储最近10条搜索记录
- ✅ **匹配高亮**: 搜索结果中关键词高亮显示
- ✅ **上下文预览**: 显示匹配内容的上下文片段

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| ⌘K / Ctrl+K | 打开搜索面板 |
| ↑ / ↓ | 选择搜索结果 |
| ↵ (Enter) | 确认选中并跳转 |
| ESC | 关闭搜索面板 |
