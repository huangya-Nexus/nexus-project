```
┌─────────────────────────────────────────┐
│  🌐 地址栏输入：code.visualstudio.com    │
│     然后按回车                           │
└─────────────────────────────────────────┘

```
```
┌─────────────────────────────────────────┐
```

## 🎯 第3步：安装Node.js（10分钟）

```
┌─────────────────────────────────────────┐
│  🌐 地址栏输入：nodejs.org               │
│     然后按回车                           │
└─────────────────────────────────────────┘
```

### 点击绿色的大按钮

```
┌─────────────────────────────────────────┐

│     │                         │         │

│     │    (Recommended)        │         │
│     │                         │         │
│     │    [Download]           │         │
│     │                         │         │
│     └─────────────────────────┘         │
│                                         │
└─────────────────────────────────────────┘
```

### 安装Node.js

**Mac用户：**
```
┌─────────────────────────────────────────┐
│  1. 双击下载的 .pkg 文件                 │
│  2. 一直点"继续"                         │
│  3. 输入电脑密码                         │
│  4. 安装完成                             │
└─────────────────────────────────────────┘
```

**Windows用户：**
```
┌─────────────────────────────────────────┐
│  1. 双击下载的 .msi 文件                 │
│  2. 一直点"Next"                         │
│  3. 安装完成                             │
└─────────────────────────────────────────┘
```

### 验证安装成功

**Mac用户：**
```
┌─────────────────────────────────────────┐
│  1. 按 Command + 空格                    │
│  2. 输入"终端"，回车                     │
│  3. 在终端输入：node -v                  │
│  4. 看到版本号，比如：v20.11.0           │
│  5. 成功！                               │
└─────────────────────────────────────────┘
```

**Windows用户：**
```
┌─────────────────────────────────────────┐
│  1. 按 Win + R                           │
│  2. 输入 cmd，回车                       │
│  3. 在命令行输入：node -v                │
│  4. 看到版本号，比如：v20.11.0           │
│  5. 成功！                               │
└─────────────────────────────────────────┘
```
