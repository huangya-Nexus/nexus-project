# 知链 Nexus - 小白1个月入门执行指南

## 一、先搞清楚：最终做出什么？

### 答案：先做出网页版（Web App）

```
第1个月目标：做出一个能在浏览器里打开的网站
├── 电脑端：用Chrome/Safari/Edge打开就能用
├── 手机端：手机浏览器也能打开（自适应）
└── 像APP一样：可以添加到手机桌面，离线也能用（PWA）

真正的APP（iOS/Android）
└── 第2-3个月再做，先用网页验证想法
```

### 为什么先做网页？

| 对比 | 网页版 | 真正APP |
|------|--------|---------|
| **开发速度** | 1个月搞定 | 3-6个月 |
| **成本** | ¥0-5000 | ¥2-10万 |
| **用户获取** | 分享链接就能用 | 要下载安装 |
| **修改迭代** | 随时改随时生效 | 要审核上架 |
| **验证想法** | ✅ 最适合 | 风险太高 |

**结论**：先用网页验证市场，有用户了再考虑做APP

## 四、Week 2：做出第一个功能（用户注册登录）

### 这是你的网站第一步：让用户能注册账号

**Day 8-10：做注册页面**

```
你要修改的文件：
frontend/src/pages/Register.tsx

这个页面包含：
├── 邮箱输入框
├── 密码输入框
├── 确认密码输入框
└── 注册按钮

效果：用户填完点击注册，数据保存到数据库
```

**代码长什么样（我帮你写好了，你改改文字就行）：**

```tsx
// 这是注册页面代码，复制进去就能用
import { useState } from 'react'
import { useNavigate } from 'react-router-dom'

export default function Register() {
  const [email, setEmail] = useState('')
  const [password, setPassword] = useState('')
  const navigate = useNavigate()

  const handleRegister = async () => {
    // 调用注册API
    const res = await fetch('/api/auth/register', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email, password })
    })
    
    if (res.ok) {
      alert('注册成功！')
      navigate('/login')  // 跳转到登录页
    }
  }

  return (
    <div className="register-page">
      <h1>注册知链账号</h1>
      <input 
        type="email" 
        placeholder="邮箱"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
      />
      <input 
        type="password" 
        placeholder="密码"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
      />
      <button onClick={handleRegister}>注册</button>
    </div>
  )
}
```

**预计时间**：2-3天（边学边做）  
**难度**：⭐⭐⭐（需要理解基础代码）

**这是调用的OpenAI API，代码我已经写好：**

```typescript
// 调用AI提取知识点
async function extractKnowledge(text: string) {
  const response = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'gpt-3.5-turbo',
      messages: [{
        role: 'user',
        content: `从以下文本中提取知识点，返回JSON格式：\n\n${text}`
      }]
    })
  })
  
  return response.json()
}
```
