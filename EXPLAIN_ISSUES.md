# Nexus 项目 - 问题说明

## 1. JWT 密钥（安全相关）
**是什么？**
JWT (JSON Web Token) 是用户登录后的身份凭证，密钥用来验证这个凭证是否合法。

**当前问题：**
密钥用的是默认值 "your-production-jwt-secret-key-change-this"

**风险：**
- 别人知道这个密钥就能伪造登录凭证
- 可以冒充其他用户登录
- 数据不安全

**解决方法：**
生成一个随机强密钥替换默认值

---

## 2. 前端 dist（构建输出）
**是什么？**
dist 目录是前端代码编译后的静态文件，用于生产环境部署。

**当前问题：**
dist 目录是 3月31日的旧版本，不包含今天新增的文件上传功能。

**影响：**
- 如果直接部署到服务器，新功能不会生效
- 用户看不到最新的界面

**解决方法：**
重新运行 npm run build 生成最新版本

---

## 3. CORS 配置（跨域访问）
**是什么？**
CORS 控制哪些网站可以访问后端 API。

**当前问题：**
配置为 "https://your-domain.com"（生产环境）

**影响：**
- 本地开发时前端无法连接后端
- 浏览器会报跨域错误

**解决方法：**
本地开发改为 "*" 或 "http://localhost:5173"

---

## 4. OSS 未启用（文件存储）
**是什么？**
OSS (对象存储) 用来存放上传的文件，比如用户上传的 PDF、图片等。

**当前问题：**
没有配置阿里云 OSS 密钥

**影响：**
- 文件只能存在服务器本地
- 服务器硬盘满了就无法上传
- 文件没有备份，服务器坏了文件就丢了

**解决方法：**
配置阿里云 OSS 密钥，文件存到云端

---

## 5. 学习连续天数（显示问题）
**是什么？**
统计用户连续学习的天数，用于激励。

**当前问题：**
有学习记录但显示 0 天

**影响：**
- 只是显示问题，不影响功能
- 用户体验稍差

**解决方法：**
检查时区设置和计算逻辑

---
